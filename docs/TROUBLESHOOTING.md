# Troubleshooting

## Problemas Comuns

### Assets não carregam
- Verifique se os caminhos estão corretos
- Certifique-se de que os arquivos existem
- Use ferramentas de desenvolvedor para ver erros 404

### Layout quebrado
- Verifique se o CSS está carregando
- Teste em diferentes navegadores
- Use media queries para responsividade

### JavaScript não funciona
- Verifique console do navegador
- Certifique-se de que o JS está carregando
- Teste funcionalidades individualmente

### SEO não funciona
- Valide meta tags
- Submeta sitemap.xml ao Google Search Console
- Verifique robots.txt

### Performance lenta
- Otimize imagens
- Minifique CSS/JS
- Use cache headers
- Reduza requests HTTP

## Ferramentas de Debug

- Chrome DevTools
- Firefox Developer Tools
- Lighthouse
- WebPageTest
- GTmetrix

## Logs

- Verifique logs do servidor
- Use Google Analytics para erros
- Monitore uptime
