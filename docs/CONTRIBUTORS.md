# Contribuidores

## Mantenedores

- **GDZ Consultoria** - Desenvolvimento inicial

## Como se tornar contribuidor

1. Fork o projeto
2. Faça suas melhorias
3. Abra um Pull Request
4. Aguarde revisão

## Tipos de Contribuição

- **Código**: Correções, melhorias, novas funcionalidades
- **Documentação**: Guias, tutoriais, exemplos
- **Design**: UI/UX, acessibilidade
- **Testes**: Casos de teste, relatórios de bug
- **Tradução**: Idiomas adicionais

## Reconhecimento

Todos os contribuidores são listados aqui e no arquivo `humans.txt`.

## Comunicação

- Issues no GitHub para discussões
- Email para questões privadas
- WhatsApp para contato rápido

Obrigado a todos os contribuidores!
