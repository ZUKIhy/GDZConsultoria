# Glossário

## Termos Técnicos

**CSS Minificado**: Versão comprimida do CSS para carregamento mais rápido.

**Favicon**: Ícone que aparece na aba do navegador.

**Git**: Sistema de controle de versão.

**GitHub Pages**: Hospedagem gratuita para sites estáticos.

**HTML Semântico**: Uso correto de tags HTML para significado.

**JS Minificado**: Versão comprimida do JavaScript.

**Manifest.json**: Arquivo para Progressive Web App (PWA).

**Meta Tags**: Informações sobre a página para buscadores.

**Open Graph**: Protocolo para compartilhamento em redes sociais.

**PWA**: Aplicativo web progressivo.

**Responsive Design**: Design que se adapta a diferentes telas.

**Robots.txt**: Arquivo que instrui crawlers de buscadores.

**Schema.org**: Vocabulário para marcação de dados estruturados.

**Security.txt**: Arquivo para relatórios de segurança.

**SEO**: Search Engine Optimization (otimização para buscadores).

**Sitemap.xml**: Mapa do site para crawlers.

**WCAG**: Web Content Accessibility Guidelines.

**WebP**: Formato de imagem moderno e comprimido.
