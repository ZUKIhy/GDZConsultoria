# Guia de SEO

## Meta Tags

Cada página HTML inclui:

- `<title>` descritivo
- `<meta name="description">`
- `<meta property="og:*">` para Open Graph
- `<link rel="icon">` para favicon

## Arquivos de SEO

- `sitemap.xml`: lista todas as páginas
- `robots.txt`: instrui crawlers
- `humans.txt`: informações sobre o site

## Otimizações

- URLs amigáveis
- Heading hierarchy (H1, H2, etc.)
- Alt text em imagens
- Links internos
- Carregamento rápido (CSS/JS minificados, imagens otimizadas)

## Ferramentas

- Google Search Console
- Google Analytics
- PageSpeed Insights
- Lighthouse
