# Guia de Acessibilidade

## Conformidade

O site visa conformidade com WCAG 2.1 AA.

## Implementações

### HTML
- Semântico (header, nav, main, section, footer)
- Labels em formulários
- Alt text em imagens
- Heading hierarchy

### CSS
- Contraste de cores adequado
- Focus visible
- Responsive design

### JS
- Navegação por teclado
- ARIA labels onde necessário

## Testes

- Use ferramentas como:
  - WAVE
  - axe DevTools
  - Lighthouse Accessibility
  - Screen readers (NVDA, JAWS)
